pdfPageMarker.pattern Property

Marker's pattern. In the pattern there are two simbols: #n# (that represents the actual page) and #N# (that represents the number of pages). The Default pattern is : "Page #n# Of #N#"

public string pattern {get; set;}

Example

This example shows how to use the pattern property: 

pdfPageMarker marker = new pdfPageMarker(400,30);
marker.pattern = "Page #n#/#N#";
myDoc.pageMarker = marker
......
......
The result on the document is : "Page 1/2"
